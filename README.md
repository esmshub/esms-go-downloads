# ESMS Go Downloads

This repository contains the official release downloads for **ESMS Go**.

ESMS Go is a modern reimplementation of the ESMS+ match engine, designed to simplify league management while improving performance and extensibility.

Key improvements include:

- 🖥️ A single executable replaces the traditional collection of utilities.
- ⚡ Faster execution through fixture parallisation.
- 🔧 Modern configuration and improved usability.
- 🧩 Extensible architecture with new engine capabilities.

### Source code

The ESMS Go source code is currently private while development continues. This repository is used exclusively to distribute official releases.

The long-term intention is to release the project as open source once it reaches a suitable stage of maturity.
